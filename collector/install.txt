Install Notes
Application Installation directory: C:\Progra~1\CollectService\
Must include: thresholds.cfg and settings.cfg

For PyInstaller: 
Compile: pyinstaller -F --hidden-import=win32timezone collect_service.py
Install: C:\Progra~1\CollectService\collect_service.exe install

For Python:  
sc create CollectService binPath="C:\Python\python.exe C:\Progra~1\CollectService\collect_service.py" type=own error=ignore start=auto
