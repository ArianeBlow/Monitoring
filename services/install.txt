Install Notes

Application Installation directory: C:\Progra~1\MonitoringService\

Must include: settings.cfg and certificate files



For PyInstaller: 

Compile: pyinstaller -F --hidden-import=win32timezone collect_service.py

Install: C:\Progra~1\MonitoringService\collect_service.exe install

Compile: pyinstaller -F --hidden-import=win32timezone event_service.py

Install: C:\Progra~1\MonitoringService\event_service.exe install



For Python:  

sc create CollectService binPath="python.exe C:\Progra~1\MonitoringService\collect_service.py" type=own error=ignore start=auto

sc create EventService binPath="python.exe C:\Progra~1\MonitoringService\event_service.py" type=own error=ignore start=auto
